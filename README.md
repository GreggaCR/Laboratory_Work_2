# Лабораторная работа №2: Решение задач с помощью обыкновенных дробей

## Описание задачи
Целью данной работы является разработка класса **«Дробь»** для выполнения вычислений с произвольной точностью. В основе реализации лежит использование целочисленного типа данных `mpz_t` из библиотеки **GMP**.

Программа позволяет работать с обыкновенными дробями без потери точности, которая неизбежна при использовании стандартных типов `float` или `double`.

## Реализованный функционал
Класс включает в себя следующие возможности:
* **Арифметические операции**: Сложение, вычитание, умножение и деление обыкновенных дробей.
* **Математическая оптимизация**: Метод автоматического сокращения дроби (поиск НОД).
* **Форматирование**: 
    * Вывод дроби в консоль и в файл.
    * Преобразование обыкновенной дроби в десятичный формат.
* **Тестирование**: Проверка работоспособности класса на реальных вычислительных задачах.

## Математическая база
Для каждой дроби вида $\frac{a}{b}$ реализованы правила:
1. Сложение: $\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$
2. Сокращение: $\frac{a}{b} = \frac{a / \text{gcd}(a,b)}{b / \text{gcd}(a,b)}$
